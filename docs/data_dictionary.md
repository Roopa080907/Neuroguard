# OASIS MRI Dataset Data Dictionary

## Classes

### NonDemented
Healthy brain MRI scans.

### VeryMildDemented
Very early signs of cognitive decline.

### MildDemented
Noticeable cognitive decline patterns.

### ModerateDemented
Moderate stage dementia-related changes.
